I am a long time coder

My goal is to learn as much about coding as I possibly can

My name is Daniel Hoffmann. I started learning about programming when I was 13 years old. I like to learn about odd and unusual aspects of coding, like declaritive programming languages. The Haskell programming language is one of my more preferred programming languages.

When I code, I prefer to be as modular and upgradable as I possibly can. I will sometimes spend a couple days reworking some code until I can easily add things to it and change things easily. I quite like Data-Driven Programming, but I am not that good at the text parsing generally required for it.

A lot of the coding that I have studied so far relates to making video games. I don't seem to have enough motivation to actually manage to get them to a playable state. I generally spend more time getting everything made in a flexible manner over completing the game and getting into the playable part.

I am interested in Parallel computing and want to learn more about how to connect systems together for such a thing. I would like to make a fault-tolerant system in place with enough speed for real-time execution. An online game that got faster with more people connected would be a good proof-of-concept.
